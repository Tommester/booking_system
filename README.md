# Foglalási rendszer – Frontend

Technológiák:

- React 18
- TypeScript
- Vite
- React Router DOM
- Tailwind CSS
- Axios alapú API hívások

A backend külön repóban található (Fastify + PostgreSQL):

https://github.com/horvathbencetibor/booking_system_be/tree/dev

A frontend alapértelmezés szerint a következő API URL-re csatlakozik:

.env
VITE_API_URL=http://localhost:3000
